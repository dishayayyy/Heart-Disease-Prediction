## Exploratory Data Analysis

A comprehensive exploratory data analysis was performed on a clinical heart disease dataset to understand feature distributions, risk patterns, and relationships with disease presence.

**Key highlights:**
- Target variable distribution and class imbalance analysis
- Univariate analysis of clinical indicators
- Bivariate analysis linking symptoms and diagnostics to heart disease
- Correlation analysis to assess feature relevance and multicollinearity
